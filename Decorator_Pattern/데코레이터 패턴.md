# 데코레이터 패턴

- `동적`으로 `책임 추가`가 필요할 때 `데코레이터 패턴`을 사용 할 수 있다

#### 예제
**커피 제조 방법**
- 에스프레소 : 커피의 기본
- 아메리카노 : 에스프레소 + 물
- 카페라떼 : 에스프레스 + 스팀밀크
- 헤이즐넛 : 아메리카노 + 헤이즐넛 시럽
- 카페모카 : 카페라떼 + 초콜릿
- 캬라멜 마끼아또 : 카페라떼 + 캬라멜 시럽

#### 디자인 원칙
- OCP(Open-Closed principle)
    - 클래스는 확장에 대해서는 열려 있어야하지만<br>
    코드 변경에 대해서는 닫혀 있어야 한다.
